# Bizarre Desktop

Bizarre Desktop es una wiki de personajes y elementos de JoJo's Bizarre Adventure presentada como si fuera un escritorio de ordenador. Simula la experiencia de entrar y navegar como si estuvieras utilizando un sistema operativo.

## Descripción

Este proyecto ofrece acceso a información sobre JoJo de forma interactiva y visual, con una interfaz que imita un entorno de escritorio para mejorar la experiencia del usuario.

## Instalación

1. Clona el repositorio:
  git clone https://github.com/N03XCUS3S/DesktopWeb.git

3. Abre el proyecto en un servidor local (puedes usar extensiones como Live Server en VS Code).

4. Abre el navegador en `index.html` para comenzar a usar la wiki.

## Uso

- Navega por la interfaz simulada de escritorio.
- Abre las diferentes ventanas para consultar información sobre personajes, Stands y otras características de JoJo.
- Explora la wiki y descubre los contenidos disponibles.
