# Bizarre Desktop

Bizarre Desktop es una wiki interactiva de personajes y elementos de **JoJo's Bizarre Adventure** presentada como un escritorio de ordenador retro. Simula la experiencia de navegar por un sistema operativo mientras exploras el universo de JoJo.

## Descripción

Este proyecto ofrece acceso a información completa sobre las partes de JoJo's Bizarre Adventure de forma interactiva y visual. La interfaz imita un entorno de escritorio clásico, creando una experiencia única para los fanáticos de la serie.

### Características

- 📚 **Wiki**: Información sobre todas las partes del manga/anime
-  **Base de datos de personajes**: Personajes con detalles como color de cabello, ojos y habilidades Hamon
- ⚡ **Stands**: Catálogo de todos los Stands con sus tipos y usuarios
- 🖥️ **Interfaz de escritorio**: Simulación de un sistema operativo retro
- 🎮 **Modelo 3D interactivo**: Portátil 3D en la página de inicio

## Partes Disponibles

El proyecto incluye información de las 9 partes de JoJo's Bizarre Adventure:

1. **Phantom Blood** - La primera parte, ambientada en la Inglaterra victoriana
2. **Battle Tendency** - La segunda parte, con los Hámones y los Popes
3. **Stardust Crusaders** - La tercera parte, el viaje a Egipto
4. **Diamond is Unbreakable** - La cuarta parte, Morioh y Kira
5. **Vento Aureo** - La quinta parte, la mafia italiana
6. **Stone Ocean** - La sexta parte, la prisión Green Dolphin Street
7. **Steel Ball Run** - La séptima parte, la carrera a través de América
8. **JoJolion** - La octava parte, el nuevo Morioh
9. **The JoJoLands** - La novena parte, Hawái

## Tecnologías

- **HTML5** - Estructura de las páginas
- **CSS3** - Estilos y diseño responsive
- **JavaScript** - Interactividad y lógica
- **Google Model Viewer** - Modelo 3D del portátil

## Instalación

1. Clona el repositorio:
   
```
bash
   git clone https://github.com/N03XCUS3S/DesktopWeb.git
   
```

2. Abre el proyecto en un servidor local (puedes usar la extensión Live Server de VS Code).

3. Abre `index.html` en tu navegador para comenzar.
